## ✨ Fitur Utama

- **Clean Home Screen** — Jam, tanggal, dan cuaca dengan tipografi premium  
- **Smart App Access** — Aplikasi yang dipin atau paling sering digunakan langsung di home  
- **Lightning Fast Drawer** — Daftar alfabet + pencarian langsung (real-time)  
- **Productivity Feed** — Event kalender, daftar tugas, dan insight screen time  
- **Full Transparent Support** — Cocok dengan live wallpaper & background transparan  
- **Ringan & Cepat** — Performa tinggi, izin minimal, tidak boros baterai  
- **Bilingual** — Bahasa Indonesia & English

## 🚀 Cara Install (30 Detik)

1. **Download** APK terbaru di [Releases](https://github.com/01satria/LauncherAppv3/releases/latest)
2. Izinkan instalasi dari "Sumber Tidak Dikenal"
3. Buka aplikasi → **Set as Default Launcher**

> **Tips:** Tekan tombol **Home** dua kali untuk kembali ke Flow kapan saja.
